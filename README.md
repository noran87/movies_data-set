#  Movie Data Analysis & Machine Learning Portfolio

This repository contains a collection of projects focused on exploring movie datasets using Python, data analysis, and machine learning techniques.

The goal of this repository is to demonstrate a clear learning progression:
- Data cleaning and preparation
- Exploratory data analysis (EDA)
- Recommendation systems
- Machine learning prediction
- Model improvement and evaluation

---

##  Project 1: IMDB Top 1000 Analysis
Explored relationships between ratings, votes, and genres.
Tools: Pandas, Matplotlib, Seaborn

---

##  Project 2: Rule-Based Movie Recommendation System
Built a rule-based recommender using genre and rating logic.
Tools: Pandas, NumPy

---

##  Project 3: ML Movie Recommendation System
Content-based recommender using TF-IDF and cosine similarity.
Tools: Pandas, Scikit-learn

---

##  Project 4: Movie Rating Predictor v1
Logistic Regression model to classify high-rated movies.
Accuracy: ~65%

---

##  Project 5: Movie Rating Predictor v2
- Added more features (genre, director, actors)
- Compared multiple models

---

##  Project 6: Movie Rating Predictor v3
- Feature engineering (grouping rare categories)
- Log transformation of votes
- Model comparison + cross-validation

Result:
- Random Forest performed best
- Stable performance across folds
